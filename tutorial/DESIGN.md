# Design

## Style Prompt

Technical editorial tutorial with a dark ink canvas, crisp code-like panels, bright confirmation accents, and chart-forward explanation. The mood should feel practical, senior, and implementation-ready.

## Colors

- Canvas: `#101418`
- Panel: `#182026`
- Text: `#F2F5F7`
- Muted text: `#A9B3BD`
- Confirmation green: `#44D492`
- Alert amber: `#F0B84A`
- Graph cyan: `#4DB7E8`

## Typography

- Headings: `Inter`, `Arial`, sans-serif
- Code labels: `IBM Plex Mono`, `SFMono-Regular`, monospace

## What NOT to Do

- No generic blue-purple gradient background.
- No decorative blobs or soft bokeh.
- No tiny unreadable code dumps.
- No marketing hero cards.
